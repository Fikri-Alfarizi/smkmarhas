# Penjelasan Website LPK Cahaya Tanu Sudrajat

## 1. Gambaran Umum Website

Website **LPK Cahaya Tanu Sudrajat (LPK CTS)** merupakan website **company profile dan sistem informasi** untuk sebuah lembaga pelatihan kerja yang berfokus pada **persiapan karier dan studi ke Jepang**, khususnya melalui program SSW (Tokutei Ginou), magang, kursus bahasa Jepang, serta program caregiver (Kaigo).

Website ini **bukan website sekolah formal**, melainkan website **lembaga pelatihan kerja profesional** yang dirancang untuk:

* Menyampaikan informasi resmi lembaga
* Mempromosikan program pelatihan
* Menampilkan dokumentasi kegiatan dan alumni
* Menyediakan artikel edukatif seputar karier ke Jepang
* Menjadi media komunikasi antara calon peserta dan pihak LPK

Website dibangun menggunakan **Laravel (backend)** dan **HTML, CSS, JavaScript (frontend)** dengan konsep **CMS (Content Management System)** sehingga seluruh konten dapat dikelola oleh admin.

---

## 2. Tujuan Pembuatan Website

Tujuan utama website ini adalah:

1. Menjadi identitas digital resmi LPK Cahaya Tanu Sudrajat
2. Memberikan informasi yang jelas, terstruktur, dan terpercaya
3. Memudahkan calon peserta memahami alur program kerja ke Jepang
4. Menampilkan kredibilitas lembaga melalui galeri, artikel, dan profil
5. Menjadi sistem manajemen konten yang fleksibel dan mudah dikembangkan

Website ini dibuat menyerupai **website perusahaan / lembaga profesional**, bukan website pembelajaran internal sekolah.

---

## 3. Struktur Halaman Website

### a. Halaman Home

Halaman utama berisi:

* Header (logo, navigasi, kontak)
* Slider / banner utama
* Informasi singkat tentang LPK
* Daftar program unggulan
* Keunggulan lembaga
* Ajakan bergabung (Call to Action)
* Galeri singkat
* Footer informasi

Fungsi halaman ini adalah memberikan **kesan pertama yang profesional** dan menarik bagi pengunjung.

---

### b. Halaman Tentang Kami

Berisi:

* Profil lengkap LPK Cahaya Tanu Sudrajat
* Visi lembaga dalam mencetak SDM siap kerja
* Pengalaman dan legalitas lembaga
* Keunggulan (trainer profesional, kurikulum, jaminan)

Halaman ini berfungsi untuk **membangun kepercayaan (trust)**.

---

### c. Halaman Program Kami

Menampilkan daftar program pelatihan, seperti:

* Program SSW (Tokutei Ginou)
* Program Reguler
* Persiapan Magang
* Kursus Bahasa Jepang
* Program Beasiswa Caregiver (Kaigo)

Setiap program memiliki:

* Deskripsi
* Manfaat
* Alur program
* Persyaratan

Data program diambil dari **database**, sehingga dapat diubah melalui admin.

---

### d. Halaman Detail Program (Contoh: Caregiver)

Halaman ini berisi informasi detail satu program tertentu, meliputi:

* Penjelasan program
* Program highlights
* Persyaratan peserta
* Alur belajar hingga bekerja di Jepang

Halaman ini bersifat **dinamis** berdasarkan data database.

---

### e. Halaman Galeri

Menampilkan dokumentasi:

* Kegiatan pelatihan
* Aktivitas kelas
* Alumni

Galeri disusun berdasarkan **album (galleries)** dan **foto (photos)**, sehingga rapi dan terstruktur.

---

### f. Halaman Blog & Artikel

Berisi artikel edukatif seperti:

* Tips interview kerja ke Jepang
* Perbedaan SSW dan magang
* Budaya kerja Jepang
* Kisah sukses alumni

Artikel ditulis oleh admin dan dikelompokkan berdasarkan kategori.

---

### g. Halaman Kontak

Berisi:

* Alamat lembaga
* Nomor telepon & WhatsApp
* Email
* Form kirim pesan

Pesan yang dikirim pengunjung akan tersimpan di database untuk dikelola admin.

---

## 4. Struktur Database dan Model

Website ini menggunakan beberapa tabel utama:

### a. Users

Digunakan untuk akun admin yang mengelola website.

### b. Posts & Categories

Digunakan untuk sistem blog/artikel dengan relasi kategori dan penulis.

### c. Programs

Menyimpan data program pelatihan, termasuk manfaat dan alur program dalam format JSON.

### d. Galleries & Photos

Digunakan untuk galeri foto dengan sistem album agar data lebih rapi dan efisien.

### e. Contact Submissions

Menyimpan pesan dari pengunjung website melalui halaman kontak.

Struktur database ini dirancang agar:

* Mudah dikembangkan
* Aman
* Fleksibel untuk kebutuhan konten jangka panjang

---

## 5. Teknologi yang Digunakan

* **Backend:** Laravel
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MySQL
* **Arsitektur:** MVC (Model View Controller)
* **Keamanan:** Auth Laravel, CSRF Protection

Website ini **bukan template statis**, melainkan website dinamis berbasis database.

---

## 6. Perbedaan dengan Website Sekolah

Perbedaan utama website ini dengan website sekolah adalah:

* Fokus pada **promosi lembaga dan layanan**, bukan akademik
* Tidak memiliki fitur nilai, kelas, atau siswa
* Mengutamakan branding, kepercayaan, dan informasi publik
* Bersifat **company profile + CMS**

Website ini lebih mendekati **website perusahaan, LPK, atau startup edukasi**.

---

## 7. Kesimpulan

Website LPK Cahaya Tanu Sudrajat merupakan website profesional berbasis Laravel yang dirancang untuk mendukung aktivitas promosi, informasi, dan komunikasi lembaga pelatihan kerja. Website ini memiliki struktur yang rapi, database terorganisir, dan sistem manajemen konten yang fleksibel sehingga dapat digunakan dalam jangka panjang dan dikembangkan sesuai kebutuhan.

Website ini dibuat sebagai **proyek mandiri non-sekolah** dan mencerminkan kemampuan pengembang dalam membangun sistem web skala nyata.

---

### 8.2 Penjelasan Singkat Struktur

* **app/Http/Controllers/Admin** → controller khusus admin (CMS)
* **app/Http/Controllers/Frontend** → controller halaman publik
* **public/assets** → file CSS, JS, dan gambar statis
* **resources/views** → blade template frontend & admin
* **storage/app/public** → file upload dari admin (gambar, galeri)
* **routes/admin.php** → routing khusus admin panel

Struktur ini dibuat agar kode **rapi, scalable, dan mudah dikembangkan** di masa depan.

Program Beasiswa Caregiver (Kaigo)
Program Beasiswa Caregiver (Kaigo) adalah salah satu program unggulan LPK Cahaya Tanu Sudrajat yang membuka kesempatan bagi peserta untuk meniti karier di bidang perawatan lansia (caregiving) di Jepang melalui jalur resmi dan terstruktur.

Program ini memberikan beasiswa pelatihan bahasa Jepang, budaya kerja, serta keterampilan caregiving bagi peserta berprestasi yang berminat bekerja di sektor kesehatan dan kemanusiaan.

Program ini memberikan beasiswa bahasa jepang, budaya kerja, serta pendidikan caregiver setara D3 (SENMON GAKKOU)

Program Highlights
Pelatihan Bahasa Jepang Intensif (Level N5–N3)
Pelatihan Keterampilan Caregiving sesuai standar Jepang
Pemahaman Budaya & Etika Kerja Jepang di sektor kesehatan
Pendampingan ujian sertifikasi dan proses seleksi kerja ke Jepang
Bimbingan administrasi hingga keberangkatan ke Jepang
Selain mendapatkan ilmu dan pengalaman berharga, peserta juga berkesempatan memperoleh pekerjaan bergengsi di Jepang dengan penghasilan stabil dan lingkungan kerja profesional.

Persyaratan Kaigo
Lulusan SLTA Sederajat, D3/S1
Laki-laki / perempuan berusia 18 - 30 tahun
Memiliki kemampuan B. Jepang minimal JLPT N3
Sehat Jasmani dan Rohani

<?php

namespace App\Models;

// use Illuminate\Contracts\Auth\MustVerifyEmail;
use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Foundation\Auth\User as Authenticatable;
use Illuminate\Notifications\Notifiable;

class User extends Authenticatable
{
    /** @use HasFactory<\Database\Factories\UserFactory> */
    use HasFactory, Notifiable;

    /**
     * The attributes that are mass assignable.
     *
     * @var list<string>
     */
    protected $fillable = [
        'name',
        'email',
        'password',
    ];

    /**
     * The attributes that should be hidden for serialization.
     *
     * @var list<string>
     */
    protected $hidden = [
        'password',
        'remember_token',
    ];

    /**
     * Get the attributes that should be cast.
     *
     * @return array<string, string>
     */
    protected function casts(): array
    {
        return [
            'email_verified_at' => 'datetime',
            'password' => 'hashed',
        ];
    }
}

<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;

class Post extends Model
{
    use HasFactory;
    
    // --- TAMBAHKAN INI ---
    protected $fillable = [
        'category_id', 
        'author_id', 
        'title', 
        'slug', 
        'excerpt', 
        'content', 
        'featured_image', 
        'status', 
        'published_at'
    ];
    // ---------------------

    // Relasi ke Kategori
    public function category()
    {
        return $this->belongsTo(Category::class);
    }

    // Relasi ke Penulis (User Admin)
    public function author()
    {
        return $this->belongsTo(User::class, 'author_id');
    }
}
